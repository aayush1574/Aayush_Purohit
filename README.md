# Aayush Purohit — 3D Portfolio

Personal portfolio site for **Aayush Purohit** — Computer Science Engineer
focused on AI Engineering, Data Analytics and Business Intelligence.

Built with React, TypeScript, Vite, Three.js / React-Three-Fiber and GSAP.
Includes an interactive 3D keyboard "Tech Stack" section where every keycap
is one of my skills, plus animated landing, career timeline, work showcase,
and contact sections.

## Tech Stack

- **Core:** React 18, TypeScript, Vite
- **3D & Animation:** Three.js, `@react-three/fiber`, `@react-three/drei`, `@react-three/rapier`, `@react-three/postprocessing`, GSAP, `@gsap/react`
- **UI:** `react-icons`, `react-fast-marquee`
- **Analytics:** `@vercel/analytics`

## Getting Started

```bash
git clone https://github.com/aayush1574/Aayush_Purohit.git
cd Aayush_Purohit
npm install
npm run dev
```

The dev server runs on [http://localhost:5173](http://localhost:5173).

## Scripts

| Script | What it does |
|---|---|
| `npm run dev` | Start the Vite dev server |
| `npm run build` | Type-check and build for production into `dist/` |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Run ESLint |
